### **ℹ️ Tentang Mata Kuliah**

**Deskripsi**  
Mata kuliah ini membahas konsep-konsep dasar kecerdasan buatan (Artificial Intelligence), meliputi teknik-teknik machine learning, deep learning, dan penerapannya dalam berbagai kasus nyata. Mata kuliah dirancang untuk mahasiswa yang ingin memahami dan menerapkan algoritma kecerdasan buatan dalam menyelesaikan permasalahan praktis. Selain aspek teoretis, mata kuliah ini juga mencakup hands-on praktis menggunakan tools dan library populer untuk implementasi model AI.

### Capaian Pembelajaran

1. **Machine Learning Fundamentals**  
    Mahasiswa mampu memahami dan mengimplementasikan algoritma machine learning dasar menggunakan Python, termasuk supervised dan unsupervised learning untuk berbagai aplikasi.

2. **Deep Learning & Computer Vision**  
    Mahasiswa mampu merancang dan mengimplementasikan model deep learning, khususnya untuk aplikasi computer vision seperti deteksi objek, klasifikasi gambar, dan pengenalan pola.

3. **AI Model Deployment**  
    Mahasiswa mampu mengembangkan pipeline lengkap untuk training, evaluasi, dan deployment model AI dalam aplikasi praktis.
